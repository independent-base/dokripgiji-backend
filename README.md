## 📖 프로젝트 소개

> 아무리 집 구하는 건 발품을 파는거라지만 이건 너무 심한거 아냐?

**독립기지**는 사용자의 생활패턴을 중심으로 부동산을 탐색할 수 있는 웹 서비스입니다. **사용자의 생활 거점(회사, 학교 등)** 을 중심으로, 단순 직선거리가 아닌 보행시간을 고려하여 후보지를 함께 찾아줍니다.
